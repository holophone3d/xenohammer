# XenoHammer — Game

This directory contains both implementations of XenoHammer and the shared game specification.

| Directory | Description |
|-----------|-------------|
| [`web/`](web/) | TypeScript/Canvas 2D browser rewrite (Vite + vanilla TS) |
| [`classic/`](classic/) | Original C++ source with ClanLib 0.6 API shim (SDL2 + OpenGL) |
| [`SPEC.md`](SPEC.md) | Authoritative game specification shared by both tracks |

See the [root README](../README.md) for quick-start instructions.

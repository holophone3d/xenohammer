XenoHammerGL by Brian Smith

Download count snapshot: 2545

Genre: Action/Arcade

Platform: Windows

Description

Selected as an Independent Games Festival winner, with a public note to look for XenoHammer at GDC.

The archived description positions XenoHammer as a "next generation of RPG shooters" focused on ship upgrades, campaign progression, and a difficult alien-invasion finale.

The page also explicitly calls out the Georgia Tech student team, OpenGL accelerator requirement, and ClanLib as the underlying game library.

Archived control list

  • Directional control - Arrow keys or joy pad
  • Fire - Space or joystick button 1
  • Speed setting - Q or joystick button 2
  • Power setting - W or joystick button 3
  • Armor setting - E or joystick button 4
  • Escape booster - Esc dumps RU's into engines and returns to the mother ship
